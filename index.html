Projects

Student Management System.

This project is a Student Management System designed to manage and store student credential records efficiently. It uses Python as the backend programming language and Tkinter for building an intuitive and user-friendly graphical user interface (GUI). The system securely stores student data in MySQL databases, enabling seamless data entry, retrieval, and updates. With its robust functionality and interactive design, this project simplifies the management of student information.

Othello Reversi Game.

This project is an Othello Reversi Game developed in C language with a console-based user interface. The game implements the traditional rules of Othello, featuring logic for valid moves, flipping discs, and calculating scores. The console UI provides a straightforward and interactive experience, making it easy for players to engage with the game. This project highlights proficiency in C programming and problem-solving through well-structured code.

Handwriten Digit Recognition Model.

This project involves creating a handwritten digit recognition model using machine learning. The model is trained on image data using a Convolutional Neural Network (CNN) to recognize digits accurately. For testing, the model utilizes a webcam to capture handwritten digits in real-time and predicts the corresponding output. The project evaluates the model's performance by measuring its accuracy, demonstrating proficiency in deep learning and computer vision techniques.

Github Link

Athlete Guiding Model.

This project is an athlete-guiding machine learning model designed specifically for javelin throwers. The model processes the player's pose and compares it to that of a professional athlete in the field. Based on the comparison, it provides detailed suggestions for improvements. This project demonstrates expertise in pose estimation, machine learning, and sports analytics, aiming to enhance athletic performance through data-driven insights.
