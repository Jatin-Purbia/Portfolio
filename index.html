Projects

01. Student Management System.

This project is a Student Management System designed to efficiently manage and store student credential records.
It is built using Python for the backend and Tkinter for a user-friendly graphical interface.
Student data is securely stored in MySQL databases, enabling easy data entry, retrieval, and updates. With its intuitive design and reliable functionality, the system simplifies student information management.

Github Link

02. Othello Reversi Game.

This project is an Othello Reversi Game developed in C Language with a console-based UI.
It implements the traditional rules of Othello, including logic for valid moves, disc flipping, and score calculation. The game allows two players to take turns, with the console UI providing an interactive experience.

Github Link

03. Handwriten Digit Recognition Model.

This project creates a handwritten digit recognition model using machine learning and a Convolutional Neural Network (CNN).
It captures real-time handwritten digits via webcam and predicts the corresponding output. The model's accuracy is evaluated, demonstrating skills in deep learning and computer vision.

Github Link

04. Athlete Guiding Model.

This project is a machine learning model designed to guide javelin throwers.
It processes the player's pose and compares it to that of a professional athlete, offering detailed suggestions for improvement. The model utilizes pose estimation and machine learning techniques to provide actionable feedback, aiming to enhance performance.

Github Link
